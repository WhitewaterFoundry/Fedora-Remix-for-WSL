Existing users can update immediately by following: https://www.whitewaterfoundry.com/blog/2025/05/05/fedora-remix-for-wsl-42-released

43.0.0
* Updated to Fedora 43.
* GNOME is no longer supported due to lack of Xorg support. If you upgraded from 42, uninstall GNOME and install Xfce, KDE, or LXDE instead.
* New first-run welcome message (shown once) with support links, update guidance, and WSL1 notes.
* Improved WSL1 compatibility for Fedora 43, including automatic mitigation for gdk-pixbuf/glycin issues during updates.
* Updated graphics stack setup: Mesa installation is refreshed and streamlined for Fedora 43.
* Direct3D 12 acceleration is now available on ARM64 (via an upstream Fedora change).
* Improved Windows Terminal defaults (no auto-close on exit).

42.1.0

* Upgraded to Fedora 42
* Standards-compliant environment setup: XDG directories are now automatically configured and created when missing.
* More reliable GUI sessions in WSLg: improved handling of `/run/user/$UID` and audio runtime paths, including cleanup of stale sockets.
* Better X11 fallback behavior: more robust Windows host IP detection.
* Cleaner D-Bus sessions: per-user runtime storage with tighter environment reuse for improved isolation.
* Now systemd is enabled by default, and it's the recommended mode
* Upgraded to Mesa 25.0.4
